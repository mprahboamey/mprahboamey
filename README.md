## Projects

**[PHOTEX-clean](https://github.com/mprahboamey/PHOTEX-clean)** -- Optical neural network simulator in PyTorch. Models diffractive layers, wave propagation via the Angular Spectrum Method, and optical self-attention. Benchmarks compare projected optical performance against NVIDIA H100 baselines.

**[HoloWeights](https://github.com/mprahboamey/holoweights)** -- Software experiments in tiled weight representation and memory-mapped access patterns, using holographic addressing geometry as an organizational model.
